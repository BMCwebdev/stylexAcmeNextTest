# GlobalToastProvider Component

The **GlobalToastProvider** component provides a global toast notification system using React Aria and React Stately.

To use the component, simply add **GlobalToastProvider** anywhere in your app. Then you can use the **toastQueue** to add toasts.

```jsx
import { GlobalToastProvider, toastQueue } from './GlobalToastProvider';

function App() {
  return (
    <GlobalToastProvider />
    ...
  );
}

// To add a toast. You can add this to an onPress event on a Button for example.
toastQueue.add({
  title: 'Success!',
  description: 'This is a success toast.',
  variant: 'success',
});
```
In progress:
* Animations
* Ability to choose max # of Toasts (currently set to 5)
* Ability to set priority on Toasts.
* Programatically add, close, pause, resume.
* Set a timeout to automatically close the toast after, in milliseconds.
