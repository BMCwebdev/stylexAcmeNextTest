# SideNavFooter

The `SideNavFooter` component is used to display additional footer content in the sidebar, including a help link.

## Usage

The `SideNavFooter` component expects the following props:
- `children` (optional): Additional content to display in the footer.

### Example

```jsx
import { SideNavFooter, SideNavItems, SideNavItem } from './SideNav';

const footerItems: SideNavItem[] = [
  { icon: 'HelpCircle', label: 'Help Center', to: '/help' },
];

<SideNavFooter>
  <SideNavItems navItems={footerItems} />
</SideNavFooter>
```
