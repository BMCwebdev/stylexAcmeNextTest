# Side Navigation

The `SideNav` component is used to display a vertical navigation bar.

### Example Usage

```jsx
<SideNav>
  <SideNavItems navItems={firstSet} />
  <Separator />
  <SideNavItems navItems={secondSet} />
  <SideNavFooter>
    <SideNavItems navItems={[{ icon: 'LogOut', label: 'Log Out', to: '/logout' }]} />
  </SideNavFooter>
</SideNav>
```
