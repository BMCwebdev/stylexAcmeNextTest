# SideNavItems

The `SideNavItems` component displays a list of navigation items in the sidebar. Each item includes an icon and a label.

## Usage

The `SideNavItems` component expects an `navItems` prop that consists of an array of navigation item objects. Each object should contain the following properties:
- `icon`: The icon to display (e.g., `Home`, `Gift`, `HelpCircle`, etc.).
- `label`: The label for the navigation item.
- `to`: The URL or route the navigation item links to.
- `isActive` (optional): Indicates if the navigation item is currently active.
- `isNewWindow` (optional): Indicates if the navigation item should open in a new window.

### Example

```jsx
const navItems = [
  { icon: 'Home', label: 'Home', to: '/', isActive: true },
  { icon: 'Gift', label: 'Gifts', to: '/gifts' },
  { icon: 'HelpCircle', label: 'Help', to: '/help' },
  { icon: 'LogOut', label: 'Logout', to: '/logout' },
];

<SideNavItems navItems={navItems} />
```
